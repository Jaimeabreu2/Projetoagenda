# Projeto Agenda — Frontend

Bem-vindo ao **Projeto Agenda**! Este é o frontend de uma aplicação web desenvolvida para ajudar estudantes a organizarem suas aulas, provas, trabalhos e outros compromissos acadêmicos. A interface é intuitiva, responsiva e acessível, garantindo uma experiência agradável para todos os usuários.

## Funcionalidades principais

- **Calendário interativo**: Visualize seus compromissos em diferentes modos (mês, semana, dia).
- **Criação de eventos**: Adicione eventos diretamente no calendário ou em uma tela dedicada.
- **Perfil do usuário**: Gerencie suas informações pessoais e preferências.
- **Acessibilidade**: Recursos como navegação por teclado e suporte a leitores de tela.
- **Design responsivo**: Compatível com dispositivos móveis, tablets e desktops.

## Estrutura do projeto

- **calendarComAgenda.html**: Página principal com o calendário e painel de compromissos.
- **addEventScreen.html**: Tela para criar novos eventos com formulário detalhado.
- **profile.html**: Página para gerenciar o perfil do usuário.
- **index.html**: Tela inicial de login e cadastro.
- **style.css**: Estilos globais reutilizáveis em todas as páginas.
- **accessibility.js**: Script para melhorar a acessibilidade da aplicação.

## Como usar

1. Abra o arquivo `index.html` no navegador para acessar a tela de login.
2. Após fazer login, você será redirecionado para o calendário principal.
3. Clique em um dia no calendário para criar um evento ou use o botão "+" para acessar a tela de criação.
4. Navegue entre as páginas usando a barra lateral ou os links disponíveis.

## Tecnologias utilizadas

- **HTML5** e **CSS3**: Para a estrutura e o design responsivo.
- **JavaScript**: Para interatividade e manipulação dinâmica do DOM.
- **Google Fonts**: Para uma tipografia moderna e agradável.
- **Acessibilidade**: Implementação de padrões ARIA e suporte a navegação por teclado.

---

**Nota**: Este projeto é parte de um sistema maior que inclui um backend desenvolvido em **Spring Boot**.
